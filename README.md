# Ghost Bus Detector

This is a 4-week project to detect "ghost buses" in real-time public transit data.

## Project Structure
- **backend/** → FastAPI code to fetch, process, and serve bus data
- **frontend/** → React + Leaflet map to visualize buses
- **docker/** → Docker setup for deployment
